# bashvm

bashvm is a console tool to create, modify, and manage virtual machines.

## How to use

### Cloning bashvm

apt install git -y

git clone https://github.com/babywhale321/bashvm.git

cd bashvm/

### Install required packages
bash bashvm-installer.sh

### Run bashvm.sh
bash bashvm.sh

### Recommended OS for bashvm
Debian 12, Ubuntu 22.04
