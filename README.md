# bashvm

bashvm is a console tool to manage your virtual machines. 

## How to use

### Cloning bashvm

apt install git -y

git clone https://github.com/babywhale321/bashvm.git

cd bashvm/

### Install required packages
bash bashvm-installer.sh

### Run bashvm.sh
bash bashvm.sh

### Recommended OS for bashvm
Debian 11 and 12
